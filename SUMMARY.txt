🌐 FAZER DEPLOY:

OPÇÃO 1: GitHub Pages (Automático)
- git push para GitHub
- Settings → Pages → GitHub Actions
- Deploy automático a cada push

OPÇÃO 2: Vercel (1 clique)
- npm install -g vercel && vercel
- Pronto!

OPÇÃO 3: Manual
- npm run build
- Deploy `/dist` em qualquer hosting
